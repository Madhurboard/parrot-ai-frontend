# Core AI Concepts

- **Speaker Embeddings (x-vectors):** Extracting vocal identity mathematically from short audio samples.
- **Discrete Token Synthesis:** Mapping text directly to acoustic tokens instead of relying on outdated spectrogram pipelines.
- **Semantic Text Chunking:** Intelligently splitting scripts at natural pauses (commas/periods) to prevent GPU memory overflow.
- **In-Context Learning:** Using Whisper transcription to pair reference audio with text, helping the model learn unique pronunciations dynamically.
