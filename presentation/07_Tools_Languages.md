# Technology Stack

- **Core AI Engine:** Qwen3-TTS, PyTorch, HuggingFace Transformers.
- **Audio Processing:** Librosa, OpenAI Whisper, NumPy, Soundfile.
- **Backend API:** FastAPI (Python) for asynchronous GPU orchestration.
- **Frontend UI:** Next.js (React), Tailwind CSS.
- **Database & Storage:** Supabase (PostgreSQL + S3 Buckets for audio/embeddings).
